# CrackDev Website

This is a redesign of the CrackedDev website using Next.js, shadcn/ui and using the cracked dev api for job listing.

### Prerequisites

**Node version 18.x.x**

### Cloning the repository

```shell
git clone https://github.com/trace2798/cracked_dev_site/.git
```

### Install packages

```shell
npm i
```

### Setup .env file

```
CRACKED_DEV_API_KEY=
```

### Start the app

```shell
npm run dev
```
